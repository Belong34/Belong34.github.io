神经网络训练优化

梯度下降

  • 使用全部数据,计算量大,浪费资源

    随机梯度下降

  • 每次只使用一个样本(收敛速度慢)

    mini-batch

  • 每次使用小部分数据进行训练

    普遍问题

  • 存在梯度震荡的问题
  • 存在局部最优解,鞍点

    动量梯度下降


开始时候，能积累动量，局部极值可以跳出陷阱，也可以缓解震荡

以上几种仍存在的问题:受初始学习率影响率很大,且每一个维度的学习率一样

AdaGrad算法

用梯度的累积作为更新的分母,使得可以自动调整梯度:

  • 前期,regularizer较小,放大梯度
  • 后期,regularizer较大,缩小梯度
  • 梯度随训练次数降低
  • 每个分量有不同的学习率
  • 前期学习率太大导致regularizer过于敏感
  • 后期regularizer累积值太大，提前结束训练

    RMSProp

    由累积平方梯度变成了平均平方梯度,解决了后期提前结束的问题

    Adam

    结合了动量以及Adagrad,还有校准值

一般设置:beta1=0.9,beta2=0.999,learning_rate=1e-3 OR 5e-4

自定义

可以自己设置一些函数(指数,反函数等等)

总结:

  • 对于稀疏数据,使用学习率可自适应方法
  • SGD通常训练时间更长,最终效果比较好,但需要好的初始化和learning rate,调不好的话还是用adam吧
  • 需要训练较深较复杂的网络且需要快速收敛时,用adam(设较小的初始值)
  • Adagrad RMSprop Adam在相似的情况下表现差不多

    卷积神经网络

    一般用于解决图像问题

    问题提出

    图像参数过多(容易过拟合,收敛到较差的局部极值)

问题解决

  • 参数过多->图像的区域性->局部连接
  • 特征与位置无关->参数共享
  • 卷积->size改变->padding

基础知识

  • 输出尺寸=（步长-边距）/s+1
  • 多个通道->分别求W再相加
  • 多个卷积核->提取多种特征
  • 步长：卷积核滑动的位移
  • 激活函数:单调非线性，否则多复杂的网络仍然只是个单层的网络

池化

最大值池化，平均值池化

性质

  • 常使用不重叠，不补零
  • 没有用于求导的参数
  • 步长=池化核大小
  • 作用：减小计算量，解决平移鲁棒
  • 损失了空间精度

    全连接层

    先展开，并连接到每个神经元。参数数目比较大。在其之后就没有卷积层和池化层了

    网络结构

    卷积神经网络=卷积层+池化层+全连接层
    全卷积神经网络(输出为图像)=卷积层+池化层+反卷积层(步长为小数的话就可以变大)

卷积神经网络对比

AlexNet

  • 首次使用Relu(提升训练速度)，
  • 2-GPU并行，
  • 使用了Dropout=0.5
    • 1.消除了神经单元之前的依赖，增强泛化能力,
    • 2.相当于每次都训练了子网络，相当于子网络的组合，
    • 3.相当于数据增强
    • 4.使用在全连接层上(参数多的地方)
  • 图片随机采样
  • SGD momentum = 0.9
  • Learning rate隔一定次数降低为1/10
  • 训练多个模型做投票

VGGNet

  • 更深
  • 多使用3x3卷积核(2个3x3=5x5，3个3x3=7x7)

  • 1x1卷积层可以看作是通道之间的非线性变化,并且降维的同时不损失信息
  • 每经过一个pooling，通道数目翻倍(挽回一些丢失的信息)
  • LRN(一种局部归一化,但已经过时了)
  • 训练技巧:先训练浅层网络，再训练深层网络。(多尺度输入)->随机使用不同的尺度缩放然后再输入分类器进行训练，不同尺度训练多个分类器，然后ensemble

ResNet

问题出现

问题解决

引入残差学习，参考：https://www.imooc.com/article/44047

  • 残差结构使得网络需要学习的知识变少，容易学习
  • 残差结构使得每一层的数据分布接近，容易学习

强调残差卷积，弱化全连接

InceptionNet

特点：工程优化，同样参数两更加有效率

问题提出

  • 更深的网络容易过拟合
  • 更深的网络有了更大的计算量

    V1

  • 分组卷积：一层上同时使用多种卷积核,使视野变宽,看到不同层级的特征
  • 不同组之间的feature不交叉计算，减少计算量(Alex)

v2

  • 引入了3x3的卷积核代替5x5(vgg)

V3

  • 卷积核: 3x3=1x3+3x1

V4

  • skip connection(res的残差连接)

MobileNet

深度可分离卷积，损失精度变大，但计算量大幅减小

卷积神经网络调参

网络初始化

  • 全部为0：单层网络可以(？),多层网络会梯度消失
  • 均值为0 ,方差为0.02 的正态分布初始化

如何分析初始化结果好不好:

  • 查看初始化后各层的激活值分布(在一个区间固定分布,是好的)

批归一化 (BN)

每个batch在每一层都做归一化，当数据量非常大的时候，每个batch并不能反应整个数据的分布(批与批之间的关系消失了)

且为了确保归一化能够起到作用，另设两个参数来逆归一化(模型学习)

数据增强

归一化

使得网络更容易学习 (加入非线性的因素)

图像变换

  • 翻转，拉伸，裁剪，变形->丰富数据，对现有数据再次采样

色彩变换

  • 对比度，亮度

多尺度

  • 缩放到不同大小用同样的块随机采样

调参技巧

  • 拿到更多的数据
  • 给神经网络添加层次(从简单到复杂)
  • 紧跟最新进展，使用新方法
  • 增大训练的迭代次数(到达一定程度可以适当降低学习率，然后增大次数)
  • 尝试正则化项(模型的容量和非0参数个数有关系)
  • 使用更多gpu加速训练
  • 可视化工具检查中间状态(损失(小)，梯度(小)，准确率，学习率(自适应学习率,学习率较小的时候如果想继续训练可以手动调上来))
  • 可以先在标准数据集上训练(确保不是自己数据集的问题，同时能与其他公开的模型做对比)
  • 在小数据集上过拟合(节约时间,过拟合=预期)
  • 数据集分布平衡(如正样本99%，则需改变思考问题的思路)
  • 使用预调整好的稳定模型结构
  • Fine-tunning->在预训练好的网络结构进行微调

    常见情况

没有学习

原因可能是学习率太大了

过拟合

训练速度不够快，需要调大学习率，没有收敛的迹象，还可以继续训练

参数没有初始化好

梯度加反了，优化目标设错了

图像风格转化

内容特征

图像输入到CNN得到的某一层的激活值,越低层级效果越好

  • 保持卷积神经网络的参数不变
  • 调整图像x的像素值，使之与y图像在CNN中的内容特征变小

风格特征

图像输入到CNN得到的某一层的激活值之间的关联(内容之间的共性)，越高层级效果越好

每个层级的多个卷积核下的feature map的相似度,如何再计算真实与预测的风格特征计算平方差损失，且加权平均多个风格特征

优劣

  • 慢，每次计算都要逐步求导
  • 逐步调整,产生多张效果图

循环神经网络

通常用于NLP，处理序列式问题(可能是多个输入多个输出，1Vs多,多VS1,多VS多，实时与非实时 )


变长，两输入

EMbedding

类似于one-hot，索引变成密集向量。

变长输入(padding)

  • 后面加0补齐
  • 截断
  • 合并

弊端：

信息丢失：多个embeddeding合并,padding噪声，无主次
无效计算太多，低效：太多padding
